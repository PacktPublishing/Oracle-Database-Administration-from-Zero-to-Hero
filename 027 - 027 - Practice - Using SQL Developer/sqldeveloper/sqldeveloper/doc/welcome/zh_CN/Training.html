Oracle 的 SQL 和 PL/SQL 开发培训流程
在 Oracle University 中学习

Oracle 针对 SQL 和 PL/SQL 开发提供了全面的培训资源。学习流程分为两类，分别针对 SQL 和 PL/SQL 的入门用户和有经验的用户提供有效的课程计划。这两类学习流程的课程中均使用 SQL Developer 作为数据库开发的图形工具。

在 SQL 学习流程中，您将会学习在日常工作中大量使用的基本 SQL 语句。同时还将掌握先进的知识，能够对 SQL 语句进行微调以优化性能并将分析 SQL 语句用于数据仓库。

在 PL/SQL 学习流程中，您将会学习在日常工作中大量使用的 PL/SQL 语句以及与编程相关的高级主题。您还将掌握如何编写安全的应用程序。

 

 

除了一系列课堂教学之外，Oracle 还通过 Oracle University 联机库提供了 Web 培训资源。这是一种便利的服务，其中包括一系列课题，用于培训 Oracle 的领先技术。
